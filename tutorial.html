How to transcribe the 2014 Referendum Leaflets

This is a very simple application for the crowd-sourcing of leaflets produced by the 'YES' and 'NO' groups as part of the 2014 Scottish Referendum campaign. This project is on behalf of the Scottish Political Archive, housed at the University of Stirling.

Please help us by transcribing the content of each leaflet as you read it. Please write the text under 'The body of the leaflet'.

Please also indicate whether there are any graphic elements present in the leaflet. Please note down these elements under 'The presence of graphic elements'.

If you have any comments on the work you have done or any doubts, please write them down under 'Comments'.

Finally, remember you can seek help and answers by posting questions on the Micropasts community forum.

Enjoy!
